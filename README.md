# Traffic Flow and Congestion Prediction

Machine Learning project for predicting traffic congestion using time-based and vehicle count datasets.

## Algorithms Used
- Logistic Regression
- Decision Tree
- Support Vector Machine (SVM)
- Random Forest
- Gradient Boosting

## Results
- Accuracy: 94%
- F1 Score: 93%

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Google Colab

## Objective
To predict traffic congestion patterns and support data-driven insights for smart city planning.
